Kiến trúc và lập trình CUDA cơ bản
Kiến trúc GPU gồm nhiều Streaming Multiprocessors (SM), mỗi SM chứa nhiều cores với bộ nhớ đệm riêng (shared memory) rất nhanh.
So sánh CPU - GPU:
CPU: ít lõi, xung nhanh, cache lớn, độ trễ thấp, throughput thấp.
GPU: nhiều lõi, xung chậm hơn, cache nhỏ, độ trễ cao, throughput cực cao.
Khái niệm lập trình CUDA:
Host (CPU) và Device (GPU).
Kernel CUDA là hàm chạy song song trên nhiều thread GPU.
Thread tổ chức thành Block, Block tổ chức thành Grid (khối và lưới).
Tối ưu hóa hiệu năng phụ thuộc vào việc tối ưu truy cập bộ nhớ (memory coalescing) và chia công việc hợp lý.
Các thuật ngữ quan trọng: Warp (32 threads), GEMM (General Matrix Multiplication), kernel launch, shared memory, global memory.

CPU: designed to be versatile
GPU: designed to be parallel processing
